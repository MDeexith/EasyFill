- You MUST pick an option from the provided list (return its exact `label`). Do not invent option text.
- If no option reasonably represents the intended value, return null for that field.

Output format:
- Return ONLY a valid JSON object: `{ "<fieldId>": "<exact option label>" | null }`.
- No explanation, no code fences.
